HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standared markup language for creating web pagesand web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML descripbes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of a web page semantically and originally included cues for the appearance of the document.
